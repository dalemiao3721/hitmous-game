# 打地鼠機率遊戲 UI 原型 (Mockups) 展示

本文件展示了打地鼠遊戲 (Whack-A-Mole Game) 的 4 個關鍵 UI 狀態原型。所有設計已依據您的建議，統一為 **Premium Casino / Dark Mode / Glassmorphism** 風格。

## 1. 遊戲啟動畫面 (Start Screen)
玩家設置下注與空洞數量的初始畫面。採用霓虹綠與深紫色的極簡配色。

![Start Screen](start_screen.png)

---

## 2. 遊戲進行中 - 擊中地鼠 (Mole Whacked)
展示成功擊中地鼠後的狀態。畫面上會冒出帶有金色邊框的地鼠，並即時更新「TOTAL MULTIPLIER」倍率。

![Playing Screen](playing_screen.png)

---

## 3. 敲中空洞畫面 - 遊戲結束 (Miss! Game Over)
根據新規則，打到空洞即會 Game Over。畫面中心顯示紅色霓虹的 **「MISS! GAME OVER」** 字樣，並揭曉其餘隱藏地鼠的位置。

![Bomb Screen](bomb_screen.png)

---

## 4. 結算畫面 - 成功獲利 (Cashout Success)
玩家選擇兌現時的勝利畫面。浮動的半透明卡片 (Glassmorphism) 詳細展示了本次獲利的金額。

![Cashout Screen](cashout_screen.png)
